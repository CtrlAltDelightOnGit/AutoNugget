One-time setup — skip if nugs-dl is already in your PATH

    

      
Paste this once in your Unraid terminal to install a wrapper at /usr/local/bin/nugs-dl:

      

        

          Install wrapper
          
        

        
cat > /usr/local/bin/nugs-dl << 'EOF'
#!/bin/sh
docker run --rm \
  -v /mnt/user/appdata/autonugget/config.json:/app/config.json:ro \
  -v /mnt/user/media/Nugs:/app/downloads \
  ghcr.io/ctrlaltdelightongit/autonugget:latest "$@"
EOF
chmod +x /usr/local/bin/nugs-dl

      

      
This wraps docker run --rm — Docker and the GHCR image must already be installed. Only needed once; persists across reboots.
